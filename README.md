# myssr
一套vue2的ssr demo,按照[Vue.js 服务器端渲染指南](https://v2.ssr.vuejs.org/zh/#%E4%BB%80%E4%B9%88%E6%98%AF%E6%9C%8D%E5%8A%A1%E5%99%A8%E7%AB%AF%E6%B8%B2%E6%9F%93-ssr)搭建




## 安装
```
npm install

```
## h5开发
```
npm run dev:client

http://localhost:8080/#/
```

## 起服务
```
1. 打包
npm run build
2. 启动服务
npm run server

// 3 浏览器访问
http://localhost:3000/#/
```
